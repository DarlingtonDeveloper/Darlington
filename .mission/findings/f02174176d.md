# Summary

Created `components/mc/gate-conditions-panel.tsx` — a collapsible gate conditions panel for the MC dashboard.

## Implementation Details

- **"use client"** directive for client-side interactivity (useState for collapse toggling)
- **Props:** `{ gates: Record<string, GateCriteria>, currentStage: string }` with types from `lib/mc/types`
- **Current stage** expanded by default, other stages collapsed; all toggleable
- **Criteria checklist:** ✓ in `text-emerald-400` for met, ✗ in `text-zinc-600` for unmet
- **Approval notes** shown in italic `text-zinc-400` inside a subtle green-tinted box when gate is approved
- **Status badges** using existing `Badge` component: success (approved), danger (rejected), default (pending)
- **Sorting:** current stage first, then by STAGES order from types
- **Empty states:** handled for both no gates and no criteria
- **Dark theme:** consistent with existing panels — `bg-black/20`, `border-white/[0.06]`, mono fonts, muted color palette
- **Export:** `export default function GateConditionsPanel`

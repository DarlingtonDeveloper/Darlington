# MC Dashboard Mission 3A — See Your Project

Dependency graph visualization and specs viewer for the MC dashboard on darlington.dev/mc.

## Goals
1. Interactive dependency graph showing task relationships, status, and flow
2. Specs viewer — browse and read spec files from the mission directory
3. Wire both into the existing MC dashboard trace/mission views

## Non-Goals
- Checkpoints UI (Mission 3B)
- Zones/projects management (Mission 3B)
- Stage override controls (Mission 3B)
